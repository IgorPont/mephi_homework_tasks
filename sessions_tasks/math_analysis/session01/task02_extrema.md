# Задание 2. Найти безусловные экстремумы функции (без использования библиотек Python)

Дана функция:

**u(x, y, z) = 3x^3 + y^2 + z^2 + 6xy − 2z + 1**

---

### 1) Находим критические точки

Чтобы найти экстремумы, нужно приравнять к нулю частные производные функции:

- ux = 9x^2 + 6y
- u_y = 2y + 6x
- u_z = 2z − 2

Решаем систему **ux = u_y = u_z = 0**:

Из u_z = 0 -> z = 1  
Из u_y = 0 -> 2y + 6x = 0 -> y = −3x

Подставляем y = −3x в ux = 0:  
9x^2 + 6(−3x) = 9x^2 − 18x = 9x(x − 2) = 0

=> x = 0 или x = 2

Теперь находим y и z для каждого x:

- При x = 0: y = 0, z = 1
- При x = 2: y = −6, z = 1

**Критические точки:** (0, 0, 1) и (2, −6, 1)

---

### 2) Классификация критических точек

Для этого используем **матрицу Гессе** (матрицу вторых производных):

**H(x, y, z) =**

```
| 18x   6    0 |
|  6    2    0 |
|  0    0    2 |
```

---

### Точка (2, −6, 1)

**H(2, −6, 1) =**

```
| 36   6   0 |
|  6   2   0 |
|  0   0   2 |
```

Проверим **главные миноры** (детерминанты левых верхних подматриц):

- D1 = 36 > 0
- D2 = det
  ```
  | 36  6 |
  |  6  2 |
  ```
  = 36×2 − 6×6 = 72 − 36 = 36 > 0
- D3 = 2 × D2 = 72 > 0

Все миноры положительны => матрица положительно определена.  
Значит, в точке (2, −6, 1) - **локальный минимум**.

---

### Точка (0, 0, 1)

**H(0, 0, 1) =**

```
| 0   6   0 |
| 6   2   0 |
| 0   0   2 |
```

Первый минор D1 = 0, поэтому точно определить тип не получается.  
Проверим поведение функции вблизи точки.

Подставим z = 1 и y = ax (то есть рассматриваем направление под углом a):

u(x, ax, 1) − u(0, 0, 1) = 3x^3 + (a^2 + 6a)x^2

- Если a = −3 -> (a^2 + 6a) = −9 < 0 -> функция убывает
- Если a = 0 -> u = 3x^3 -> знак зависит от x (меняется)

В одних направлениях функция растет, в других убывает => **седловая точка**.

---

### Выводы

- (2, −6, 1) - **локальный минимум**
- (0, 0, 1) - **седловая точка**
